The Last of Us Part II

Role: UI Scripter

I served as the sole UI Scripter on the team for the majority of development, and began training a second UI Scripter towards the end of the project.

I worked directly with the UI Artist to design, implement, and iterate on the UI, using both the in-house scripting language and C++. Was also been responsible for tool improvements, Photoshop / Illustrator scripting to improve the artist's workflow, and keyframe animating the majority of the UI.

Technologies: C++, Scripting (in-house racket/lisp based language), UI Editor (in-house tool), PyQt for tool-improvements, After Effects
